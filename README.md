# 🌐 Personal Portfolio Web Application (Final Project)

## 📌 Project Overview
This project is the final version of my personal portfolio web application. It combines all concepts learned throughout the course into a polished, responsive, and interactive website.

The website showcases my personal background, projects, GitHub work, and includes dynamic features such as filtering, API integration, and real-time form validation.

---

## 🚀 Features

### 🧭 Navigation Bar
- Fixed navigation bar with smooth scrolling
- Clicking the logo returns to the Home section
- Sections include: Home, About Me, Projects, GitHub, Contact

---

### 🏠 Home Section
- Personal introduction
- Profile image with animation
- Social media links (LinkedIn, GitHub)
- Downloadable CV

---

### 👤 About Me Section
- Personal description and background
- **Read More / Read Less toggle**
- Responsive layout with image and text

---

### 📂 Projects Section
- Displays multiple projects
- **Filtering by category** (Web, Database, AI)
- **Sorting by date** (Newest / Oldest)
- Dynamic UI updates using JavaScript

---

### 🔗 GitHub Projects Section
- Fetches repositories dynamically using GitHub API
- Displays project name, description, and metadata
- Includes error handling for API failures

---

### 📩 Contact Section
- Contact form with:
  - Name
  - Email
  - Phone number
  - Message
- **Real-time validation**
- Displays **error messages under each field**
- Prevents invalid submission
- Shows success message after submission

---

### 🌙 Dark / Light Mode
- Toggle button to switch themes
- User preference saved using `localStorage`
- Automatically applied on page reload

---

## 🧭 How to Use

- Use the navigation bar to move between sections  
- Click "Read More" in About Me to expand content  
- Filter and sort projects in the Projects section  
- View GitHub repositories dynamically  
- Toggle between dark and light mode  
- Fill out the contact form and receive feedback
